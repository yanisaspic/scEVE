# scEVE: ensemblist approach to single-cell RNA-seq clustering

[![Python](https://img.shields.io/badge/Python-3.10.6-3572A5.svg)](https://www.python.org/)

[![Code style: styler](https://img.shields.io/badge/code_style-styler-198CE7.svg)](https://styler.r-lib.org/index.html)

Generate scRNA-seq clustering results robust to the choice of methodology by integrating a pool of orthogonal methods iteratively.

The pool includes :
1. Seurat
2. monocle3
3. scCCESS-SIMLR
4. scCCESS-k-means

This package includes :
- interacting Python & R scripts to generate the iterative ensemblist clustering.
